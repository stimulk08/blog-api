# blog-api
## Цель проекта
 Реализовать REST API для сайта, где пользователь может создавать посты и управлять ими
## Модули
 Проект состоит из следующих частей
 * Jwt
 * Api
 * PostgreSQL
 * Swagger
## Стек технологий
 * ЯП: Typescript
 * API: Express
 * БД: Sequelize
